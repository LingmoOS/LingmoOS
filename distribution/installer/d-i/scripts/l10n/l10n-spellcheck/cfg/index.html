Spellchecking PO files

A set of scripts was written to ease the task of spotting typos in PO files; the check is performed once a day, and statistics are made available on this page.

Documentation about the spellchecker is part of the Debian Installer Internationalization and Localization Guide

Currently, you can find statistics for the following translations:

  • d-i (level1)
  • d-i (level2)
  • d-i (level3)
  • d-i (level4)
  • d-i (level5)
  • d-i manual
