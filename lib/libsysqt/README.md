# Qt Plugins

Unify Qt application style of LingmoOS.

## Dependencies

`sudo pacman -S gcc extra-cmake-modules qt5-base qt5-tools qt5-x11extras libqtxdg libdbusmenu-qt5 libxcb`

## Build

```shell
mkdir build
cd build
cmake ..
make
sudo make install
```
