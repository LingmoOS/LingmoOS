# Lingmo Boot Maker

Lingmo Boot Maker is a tool that helps user to create a bootable usb stick quick and easy, it is designed to support lingmo install iso, but it works for all ubuntu live install iso too.

![screenshot](https://raw.githubusercontent.com/lingmoos/lingmo-boot-maker/master/docs/lingmo-boot-maker.png)

### Dependencies

You can also check the "Depends" provided in the debian/control file.

### Build dependencies

You can also check the "Build-Depends" provided in the debian/control file.

## Installation

- Official binary release

The only binary release is from Lingmo OS iso for now. You can download iso from: https://www.lingmo.org/original/lingmo-boot-maker/. Or you can find binary file on the root of iso.

- Repository Install

If you are lingmo user, you can install Lingmo Boot Maker from repository:

```bash
sudo apt-get install lingmo-boot-maker
```
