### ocean-device-formatter

A simple graphical interface for creating file system in a block device.
Originally a subproject of ocean-file-manager.

### Dependencies

### Build dependencies

* pkg-config
* libx11-dev
* Qt5(>= 5.6) with modules:
  - qtbase5-dev
  - qt5-qmake
* Lingmo-toolkit(>=5.0) with modules:
  - libdtkwidget-dev
  - libdtkgui-dev
* libudisks2-qt5-dev
