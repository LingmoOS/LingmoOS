## Lingmo Tool Kit Gui

Lingmo Tool Kit (DtkGui) is the development graphical user interface of all C++/Qt Developer work on Lingmo.

You should read the <a href=doc/Specification.md>Lingmo Application Specification</a> firstly.

## Dependencies

### Build dependencies

* Qt >= 5.10

## Installation

### Build from source code

1. Make sure you have installed all dependencies.

2. Build:

```bash
mkdir build
cd build
cmake ..
make
```

3. Install:

```bash
sudo make install
```
