# Lingmo Icon Theme

Lingmo Icon Theme provides the base icon themes on Lingmo.

## Dependencies

### Build dependencies

* Qt >= 5.6

## Installation

### Build from source code

1. Make sure you have installed all dependencies.
````
sudo apt build-dep lingmo-icon-theme
````
If you need to use the designer plugin, you should also install:
````
sudo apt install qttools5-dev
````

2. Build:

````
$ make
````

3. Install:

````
$ sudo make install
````
