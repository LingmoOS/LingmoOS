# Lingmo Font Manager

Lingmo Font Manager is used to install and uninstall font file for users with bulk install function.

## Dependencies

* sudo apt install libdtkwidget-dev qttools5-dev-tools libfontconfig1-dev libfreetype6-dev libocean-file-manager-dev

## Installation

* mkdir build
* cd build
* qmake ..
* make

## Usage

* ./lingmo-font-manager
