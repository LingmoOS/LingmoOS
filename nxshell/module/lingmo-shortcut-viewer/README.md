# lingmo-shortcut-viewer

UnionTech Software Technology Co., Ltd.<>

Lingmo - Shortcut - viewer is a single process application After the lingmo shortcut viewer is started, other applications cannot start the lingmo shortcut viewer until the lingmo shortcut viewer is destroyed. When the lingmo shortcut viewer detects that the control key or the shift key is released, the lingmo shortcut viewer will exit, or click the lingmo shortcut viewer window area with the mouse.
