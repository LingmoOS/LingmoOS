# lingmo-shortcut-viewer

UnionTech Software Technology Co., Ltd.<>

lingmo-shortcut-viewer 是一个单进程应用.当lingmo-shortcut-viewer启动后，在lingmo-shortcut-viewer被销毁之前其他应用无法启动lingmo-shortcut-viewer。当lingmo-shortcut-viewer检测到有control键或者shift键被释放时候，lingmo-shortcut-viewer会退出，或者用鼠标点击lingmo-shortcut-viewer窗口区域。
