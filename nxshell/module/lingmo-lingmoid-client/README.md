# lingmo-lingmoid-client

Can be used to log in to a network account, or to log in to cloud sync

## Dependencies

lingmo-lingmoid-client base on QWebEngine, you can install build dependencies in lingmo like this:

```bash
#For release based on debian
sudo apt-get build-dep lingmo-lingmoid-client
```

## Build

`make`

## Install

`sudo make install`
