# ocean-app-services

ocean-app-services provides a service collection of OCEAN applications, including dconfig-center.

## Dependencies

### Build dependencies

* Qt (>= 5.6)
* [dtk](https://github.com/lingmoos/lingmo-tool-kit)

## Installation

### Build from source code

1. Make sure you have installed all dependencies.

2. Build:
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

3. Install:
```
$ sudo make install
```
