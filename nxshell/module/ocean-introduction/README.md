# ocean-introduction


## Dependencies

Build-Depends: debhelper (>= 9),
 pkg-config,
 cmake,
 libdtkgui-dev,
 libdtkwidget-dev,
 libdframeworkdbus-dev,
 libdmr-dev,
 libffmpegthumbnailer-dev,
 libdtkcore-dev,
 libdtkcore5-bin,
 qttools5-dev,
 libgsettings-qt-dev,
 qtmultimedia5-dev,
 libgtest-dev, 
 libgmock-dev

#### Build
- mkdir build
- cd build
- cmake ..
- make
- sudo make install
