# lingmo-lingmoid-plugin

lingmo云的控制中心插件。

## 依赖

可以使用以下的命令安装构建依赖:

```bash
#For release based on debian
sudo apt-get build-dep lingmo-lingmoid-plugin
```

## 构建

```
cmake -Bbuild
cmake --build build -j`nproc`
```
