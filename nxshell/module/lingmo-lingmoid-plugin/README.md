# lingmo-lingmoid-plugin

lingmo-lingmoid plugin for ocean-control-center.

## Dependencies

you can install build dependencies in lingmo like this:

```bash
#For release based on debian
sudo apt-get build-dep lingmo-lingmoid-plugin
```

## Build

```
cmake -Bbuild
cmake --build build -j`nproc`
```
