# LingmoOS Updator

LingmoOS system update tool.

## Dependency package

Debian/Ubuntu Dependency:
```shell
sudo apt install cmake libqapt-dev
```

## Build

```shell
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX:PATH=/usr ..
make
sudo make install
```
